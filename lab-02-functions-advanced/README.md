# lab-02-functions-advanced

**解构赋值、不定长参数（`*args`, `**kwargs`）**

## 知识点

- 解构：`a, b = (1, 2)`、`*rest, last = [1,2,3,4]`
- `*args`： positional 可变参数，打包成元组
- `**kwargs`： keyword 可变参数，打包成字典
- 组合使用：`def f(a, b, *args, **kwargs)`

## 运行

```bash
python main.py
```
