# lab-00-08-exception-handling

**异常处理：try / except / finally**

## 知识点

- `try` / `except`：捕获异常
- `except 异常类型`：捕获特定异常
- `except Exception as e`：捕获所有异常并获取错误信息
- `finally`：无论是否异常都会执行
- `raise`：主动抛出异常
- 常见异常：`ValueError`, `TypeError`, `FileNotFoundError`, `ZeroDivisionError`

## 运行

```bash
python main.py
```

## 学习目标

- 理解异常处理的重要性
- 掌握 try/except 的用法
- 学会处理常见异常
