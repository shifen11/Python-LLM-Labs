# lab-00-06-string-operations

**字符串操作：格式化、常用方法**

## 知识点

- 字符串格式化：f-string（推荐）、`format()`、`%` 格式化
- 常用方法：
  - `upper()` / `lower()`：大小写转换
  - `strip()`：去除首尾空白
  - `split()`：分割字符串
  - `join()`：连接字符串
  - `replace()`：替换
  - `find()` / `index()`：查找子串
- 字符串拼接：`+` 或 `join()`

```bash
python main.py
```
