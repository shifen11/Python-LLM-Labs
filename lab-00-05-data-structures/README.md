# lab-00-05-data-structures

**数据结构基础：列表、元组、字典、集合**

## 知识点

- **列表 list**：`[1, 2, 3]`，可变，有序
- **元组 tuple**：`(1, 2, 3)`，不可变，有序
- **字典 dict**：`{"key": "value"}`，键值对，可变
- **集合 set**：`{1, 2, 3}`，无序，不重复
- 常用操作：添加、删除、访问元素

## 运行

```bash
python main.py
```
